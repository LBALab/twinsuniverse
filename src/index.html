Welcome to the Twinsuniverse

This is the product of what once started as the LBA compendium project in December 2000, and various other projects developed separately.

The goal of this project is to make a complete documentation of the universe around the games and alternate versions of:
- Little Big Adventure 1 (Relentless)
- Little Big Adventure 2 (Twinsen's Oddysey)
- Little Big Adventure 3 (Stellar Entity Genesis)

The Twinsuniverse currently contains 246 articles and 990 files. More numbers

Latest News Update

27th February 2024

As part of an effort to involve all the community to easily update the content of this site, we are releasing a new version with markdown static pages easily editable via git repository.

This is a copy of the latest version of the Twinsuniverse, a work done by Jack McKalling to preserve Encyclopedia Twinsunica which was no longer active.

The indent of this site is to keep all known information about the Little Big Adventure series Universe in one single place.

22th October 2012

The Twinsunica now continues as the Twinsuniverse! Thanks to xesf, this new MediaWiki was installed on his server, which thanks to Jack McKalling was setup as the new Twinsunica and completely revamped with modern day standards.

Many content from the Twinsunica was imported and improved, including the metapages and templates.

3rd July 2008

A new front page has been designed by Torch, and a post about it has been submitted to the MBN Forums. Thanks to Wacko and Homeless for the image!

Various LBA-related happenings have occoured, including progress on the LBA: Prequel and LBA: The Great Rebellion projects.
